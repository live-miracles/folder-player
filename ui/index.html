Base vMix Preset

Select a base vMix preset. If you keep a file named `base.vmix` in the content or parent folder, it will be used instead. Camera input title should be `Cam` and microphone `Mic`.

Playing Folder

App automatically creates a vMix playlist with the files inside. Use numbers in the file names to order, i.e. `05 Video.mp4`. You can also use underscores to extend numbers, i.e. `06_1 Slide.png`. To layer image over audio keep the same numbers, i.e. `07 Image.png`, `07 Audio.mp3`. Add a folder with images to create a slideshow.

If folder has other subfolders with the `folder-player.txt` config file, the script will generate a preset for each subfolder.

Preview Next Script

If you prefer to use vMix directly instead of the web UI you can make use of the `PreviewNext` script. It will automatically put the next input in the preview after transition. The script is available in GitHub.
