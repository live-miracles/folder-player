Base vMix Preset

Select a base vMix preset. Camera input title should include the keyword `Cam`, for microphone `Mic`, and you can specify the black slide with keyword `Black`.

Note: You can keep a file named `base.vmix` in the content folder or the parent folder, it will use it instead of the default one.

Play Folder

Select the folder, and the App will automatically create a vMix playlist with the files inside. You can add numbers in the file names for ordering, i.e `05 Video.mp4`.

To overlay image over audio keep the same number, i.e. `07 Image.png`. `05 Audio.mp3`. If there are more then one image they will be shown as a slideshow.

There are 3 special inputs: `Cam`, `Mic`, `Black`.

  • • To enable microphone for a slide `07 Image.png` create an empty file `07 Mic.txt`.
  • • To add camera to a song `09 audio.mp3` create an empty file `09 Cam.txt`.
