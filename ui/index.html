Base vMix Preset

Select a base vMix preset. Camera input title should be `Cam` and microphone `Mic`.

Tip: Keep a file named `base.vmix` in the content or parent folder, it will be used instead.

Playing Folder

App automatically creates a vMix playlist with the files inside. Use numbers in the file names to order, i.e. `05 Video.mp4`. To layer image over audio keep the same numbers, i.e. `07 Image.png`, `05 Audio.mp3`. Add a folder with images to create a slideshow.

Tip: Files numbered as `0` or `00` will be pinned for easy access.
